## 五、建议技术选型细节

| 组件 | 推荐方案 | 理由 |
|------|---------|------|
| 数据库 | PostgreSQL 15+ | 生态成熟、JSONB 支持、国产数据库兼容性好 |
| ORM | GORM v2 | Go 社区最成熟的 ORM |
| 缓存 | Redis 7 | 会话存储 + 指标缓存 |
| 反向代理 | Nginx | 静态文件服务 + SSL 终止 |
| 配置管理 | Viper | Go 生态标准配置库 |
| 日志 | Zerolog | 结构化、高性能 |
| 迁移工具 | golang-migrate | SQL 迁移版本控制 |
| 前端状态 | Zustand | 轻量、TS 友好 |
| 前端请求 | TanStack Query | 缓存/重试/自动刷新 |
| 前端表单 | React Hook Form + Zod | 类型安全表单验证 |
